# Productivity Skills

Daily non-code workflow tools.

| Skill Name | Description |
| :--- | :--- |
| [Write a Skill](write-a-skill/SKILL.md) | Create, write, review, and improve reusable agent skills. Use when working on skill structure, progressive disclosure, bundled resources, examples, or validation. |
| [Write Story](write-story/SKILL.md) | Backlog item drafting, story splitting, acceptance criteria, and readiness review. |
| [Caveman](caveman/SKILL.md) | Ultra-compressed communication mode that drops filler, articles, and pleasantries while keeping technical accuracy. Use when the user says caveman mode, talk like caveman, use caveman, less tokens, be brief, terse mode, or invokes /caveman. |
| [Grill Me](grill-me/SKILL.md) | Plan and design pressure-testing through one-question-at-a-time review. |
| [Management Talk](management-talk/SKILL.md) | Engineering-content rewrite workflow for leadership and cross-functional audiences. |
| [Stakeholder Update](stakeholder-update/SKILL.md) | Audience-aware status updates, sprint summaries, launch notes, and risk escalations. |
